Region Settings App

Interactive prototype built from Figma design specifications

This Angular application demonstrates the conversion of a Figma design into a fully functional, interactive prototype with realistic data, state management, and modern UI patterns.

View Prototype View Dashboard View Source

Features

Interactive Components

Collapsible sections, exclusive base region selection, and real-time search filtering.

Realistic Data

30+ cities across 5 continents with authentic country flags and proper geographic organization.

Modern Architecture

Angular with TypeScript, RxJS state management, and component-based architecture.

Responsive Design

Mobile-friendly layout with breakpoints and accessibility support throughout.

Design System

SCSS variables, mixins, and consistent styling patterns following Figma specifications.

Production Ready

Clean builds, proper error handling, and optimized performance for deployment.
